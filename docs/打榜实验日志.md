## 2. 已坐实的结论

1. **推理杠杆**：满均衡 +2.30pt（#1 vs #2）、满 TTA vs 单尺度明显（#14 单尺度 GCE 拖低）。均衡强度 **0.75 与 1.0 几乎并列**（77.73 vs 77.69，差 0.04 在噪声内）→ 均衡强度不是细调点，0.75–1.0 均可。
2. **Model Soup 是最强方向**：soup_uniform/soup_sweep ≈ **77.7**，碾压单模型(76.1)、5信号(≤75.7)、长训练(≤75.7)。
3. **多样化汤走死**：soup_v2(刻意多样)76.83 < uniform 77.69；soup_v4(val-gated 扩池)val 0.9229<0.9267 未上榜。混入异轨迹 checkpoint 破坏 soup 线性连通性。
4. **长训练无效**：run60_ep60 75.62、swa_run60 75.67，均远低于冠军。轮次不是杠杆。
5. **5信号线已死**：≤75.66，停止投入。
6. **贪心汤退化**：塌缩到单模型，无用。
