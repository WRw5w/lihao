# 训练版本演进记录

> 噪声标签细粒度识别（CLIP ViT-B/32 鲁棒微调）的逐版本配方与结果变化。
> 记录截至 2026-06-17。所有版本骨干均为 CLIP ViT-B/32（赛规强制），单模型单推理。

## 0. 关键背景：指标与陷阱

- **线上真实指标**：干净、类别均衡测试集上的**整体 Top-1 Accuracy**（赛题第六节）。
- **本地验证指标**：90/10 分层划分（seed 42），两个代理：
  - `mid_03_06`：kNN 一致性 0.3–0.6 中带准确率（团队早期主指标）。
  - `noisy_all`：验证集整体准确率，**但量在脏标签上**。
- **核心教训**：⚠️ **两个本地指标都不可靠预测榜分**。`noisy_all` 在长训练后会因"记忆脏标签"而虚高，与干净榜分方向相反（见 v5）。**榜分是唯一真相。**

## 2. 推理端杠杆（不改训练，单模型合规）

针对真实指标（整体准确率）做的推理期增强：

| recipe | 做法 | 效果 |
|---|---|---|
| plain | 翻转 TTA | 基线 |
| **balanced** | logit 校正到均匀先验（测试集已知均衡） | 预测分布 std 19→1.4，改写~16%预测 |
| tta | 多尺度(448/512/576)+翻转 | — |
| **tta_balanced** | TTA + balanced | **🏆 榜单最大提分点** |
| gce / gce_balanced | GCE 鲁棒损失训练 + 上述推理 | 待榜分验证 |

**关键发现**：测试集均衡而原始预测严重偏斜（某类预测195次/应~50）→ **balanced 校正是合规免费的真杠杆**。

**核心结论**：
1. **Model Soup（同架构权重平均）是当前最强方向**，soup_uniform=**77.69** 为新基准（替代旧"76.7"）。
2. **"多样性是关键杠杆"被推翻**：soup_v2/soup_v3 这类刻意多样化的汤 **不如** run60-heavy 的 soup_uniform。均匀平均一批*强且相近*的 checkpoint，优于硬塞入弱/异质成分。
3. 单模型(76.15) < 汤(77.69)，差约 **1.5pt**——汤的增益真实且显著。
4. swa_champion(76.73)≈conservative(76.22)≈单冠军(76.15)，聚在 76 附近；**保守化训练没有突破**，SWA 与单模型同档。
5. **5 信号线彻底死亡**（≤75.66），停止投入。
6. 待出分：soup_v3(更激进多样化汤，预计 ≤soup_uniform)、6 个满TTA单模型(gce/keep85/keep95/aug06/ema9995/drecall，预计单模型档 ~76)。**截至 2026-06-17 07:00，codex 队列慢且有偏差**（走到 swa_run60、误选了单尺度 c448_gce_balanced；soup_v3 仍未上榜）。

**下一步破 77.69 = val-gated soup_v4（2026-06-17 07:00 已启动 `exp_pipelines/soup_v4.sh`）**：把 5 个新强 breadth 模型(ema9995 val0.9282 等)+drecall 加入 `soup_lab` 候选池，让 90/10 val 自动选 top-10 均匀平均（**weak 模型进不了 top-10，避免 soup_v2 掺弱模型的覆辙**）。这是"扩大强模型池"而非"刻意多样化"。判据：v4 uniform val ≥ 冠军 val 0.9267 才生成满推理提交上榜 A/B（不浪费榜位）；输出隔离 `outputs_soup_v4_*` 不动冠军。**不依赖 soup_v3 榜分**——val 对汤选择已被证有信号（正确选出 77.69 冠军）。

## 4. v5 实验：60 轮长调度（重要负结果）

**动机**：LR 是 `OneCycleLR(total_steps=epochs×steps)`，8 轮的 ep4 峰值疑为调度假象；长调度峰值或更晚更高。

**结果**：60 轮跑下，`noisy_all` 峰值后移到 **ep54=0.7941**（比 8 轮 0.7807 高 +1.34pt），top6 全在 ep41–60。**看似验证了假设。**

**但榜单打脸**：ep60 提交反而 **−1 分**。说明 `noisy_all` 的"提升"主要是**记忆脏验证标签**（过拟合噪声），不转化为干净准确率。

**定论**：✅ 调度确实决定峰位（ep4→ep8/54）；❌ 但长训练**不提升干净榜分**。**轮次不是杠杆，砍掉这条线。** 全量重训仍取 val 峰值轮（4–6轮）即可。

## 5. 当前最优组合 & 后续方向

**最优 = soup_uniform（多 checkpoint 均匀权重平均，同架构）+ 推理 tta_balanced ＝ 榜分 77.69。** 单模型最优 c448_dr_rank32_keep90 为 76.15，汤比单模型 +1.5pt。

- ❌ 已排除：512分辨率、长训练(60轮)、更高轮次、5信号线、贪心汤(退化)、刻意多样化汤(soup_v2<uniform)。
- ✅ 已坐实：满 TTA+满均衡是赢的推理；Model Soup（均匀平均强 checkpoint）是赢的方向(77.69)。
- 🔍 待榜分验证：soup_v3、6 个单模型(gce/keep85/keep95/aug06/ema9995/drecall)。
- 🎯 破 77.69 方向：以 soup_uniform 成分为基底，**增量**加入新强 checkpoint(如 ema9995)，先 90/10 val 预筛再上榜。
- 📌 纪律：往后所有实验**只认榜分**，noisy_all/mid 仅作参考；模型间比较不臆断要实测。
- 🚫 不碰：多模型集成（违规且复赛/决赛复核必被取消）；"集成"收益只用同架构权重平均(soup/SWA，合规)。
