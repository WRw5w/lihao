# 候选策略库：45 个本质不同的新策略（2026-06-18）

> 目标：跳出"frozen B/32 头部架构 + kNN去噪 + soup"撞 77.7 天花板的局面。
> 这 45 个**机制本质不同**，且**与我们已试过的不重叠**。
> 约束（全部满足）：CLIP ViT-B/32 **冻结**、PEFT、**单模型单推理**、数字标签(无类名)、~37%噪声、8GB。
>
> **已试过(不在此列)**：LoRA/FET局部分支、kNN一致性保留、教师头伪标签、5信号可靠性、软伪标签、GCE、ELR、迭代重打标、EMA、RandAug、soup/SWA、多尺度+翻转TTA、均匀均衡校正、长训练。
>
> 攻的瓶颈：①特征利用/容量(换 PEFT)②噪声(选样/损失/半监督)③特征空间用法 ④推理。
> 成本：🟢便宜(改loss/推理) 🟡中(改训练) 🔴重(大改造)。合规默认满足，⚠=需注意单推理边界。

## 优先级建议（45 选出最该先打的几枚）

- **最便宜先验证（🟢，各 1 发即可判定）**：① DoRA（换 PEFT 抬单模型上限）⑲SCE/㉑APL（换鲁棒损失）㉛Mixup（噪声平滑）。这几个改动小、各自能 A/B。
- **噪声大牌（🔴/🟡，上限最高）**：⑩DivideMix、⑱Confident Learning、㉗标签传播——若便宜的不破 77.7，这些是噪声那条线的主力。
- **特征受限假设的关键检验**：㉕kNN分类器 / ㊶多层特征融合——若连"换用特征的方式"都到不了 77.7，基本坐实冻结 B/32 特征天花板。
- **合规注意(⚠)**：㉕kNN/㊳重排/㊱TENT 涉及推理期用训练特征或在线更新，需确认仍属"单模型单推理"(用同一模型的特征+非参分类，一般可视作合规，但提交前要核对赛规措辞)。
